How to run the project : 

1 - Create a map :
    change the grid in "create_new_map.py"

2 - Change the name of the map "map0.npy" in train.py & play.py to your map :
    src/train.py -> line 66
    play.py -> line 7

3 - Use "run_commands.ipynb" :
    write in new cell :
    %run create_new_map.py 
    %run src/train.py 
    %run play.py
